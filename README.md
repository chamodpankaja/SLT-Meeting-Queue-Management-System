# SLT Interns Meeting Queue Management App

A mobile application developed for Sri Lanka Telecom (SLT) to manage intern queues for IT department meetings, aiming to reduce physical congestion outside the conference room and improve meeting efficiency.

## 📱 App Icon


<img src="meeting_qms/assets/app_icon/app_icon1.png" alt="App Icon" width="120" height="120" style="border-radius: 10px;" />


## 🚀 Features

- 📋 Intern Queue Registration  
- ⏰ Real-Time Queue Status Monitoring  
- 🔔 Notification for Turn Alerts  
- 📅 Schedule Meeting Slots  
- 🧑‍💼 Admin Panel for IT Staff  
- 🔒 Secure Authentication

## 🏗️ Tech Stack

- **Frontend:** Flutter 
- **Backend:** Firebase 
- **Database:** Firestore
- **Authentication:** Firebase Auth
